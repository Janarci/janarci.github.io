Multithreaded loading

OpenGL || C++

Solo project utilizing multithreading to load multiple 3D objects concurrently

This project can load multiple scenes, which have 3D objects stored in it, at the same time. Scenes can be loaded and unloaded which is useful for freeing up resources. Scenes can also be hidden or shown to display multiple scenes at once.
